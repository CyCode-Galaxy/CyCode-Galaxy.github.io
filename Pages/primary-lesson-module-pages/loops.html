Loops (Repeat actions)

20 min

What are Loops?

In computer programming, a loop is a sequence of instructions that is continually repeated until a certain condition is reached. Loops allow us to repeat a block of code multiple times. This is useful when you want to perform a task repeatedly without writing the same code over and over.

Fun Fact!

Did you know !!! When you brush your teeth, you use a loop: you brush each section of your mouth repeatedly for 2 minutes. Coding of the loops also work in the same way!

Types of Loops:

for loop

For loop is a control flow statement in programming that allows you to execute a block of code repeatedly based on a specified condition. It is commonly used when you know how many times you want to execute a block of code.

for (let i = 0; i < 5; i++) { console.log("Hello " + i); }
while loop

While loop is a control flow statement in programming that allows a block of code to be executed repeatedly as long as the given condition remains true. It is often used when we want to repeat a block of code till some condition is satisfied.

let i = 0; while (i < 5) { console.log("Number: " + i); i++; }
do...while loop

Do... while loop is a control flow statement in programming that executes a block of code until the given condition is satisfied. The feature of do...while loop is that unlike the while loop, which checks the condition before executing the loop, the do...while loop ensures that the code inside the loop is executed at least once, even if the condition is false from the start.

let i = 0; do { console.log("Count:" + i); i++; } while (i < 5);

What You Learned

  • Loops help you repeat actions efficiently
  • We can use the 3 different loops for different situations (for, while and do..while loops)
  • Loops can save a lot of time by automating repetitive tasks
  • Always include a way for the loop to end (like incrementing a counter)
