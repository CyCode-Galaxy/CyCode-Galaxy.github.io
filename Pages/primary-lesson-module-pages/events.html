Events (Responding to Actions)

20 min

What are Events?

In computer programming, Events let your program respond to user actions like clicks, typing or moving the mouse. They make your programs interactive!

Fun Fact!

Every time you click, scroll, or type on a webpage, you're triggering events. A single webpage can process hundreds of events every second!

Common Event Types

Click Events

Triggered when a user clicks an element

document.getElementById("btn")
.addEventListener("click", function() {
alert("Button clicked!");
});
Keyboard Events

Triggered when keys are pressed

document.addEventListener("keydown",
function (event) {
console.log("Key pressed:", event.key);
});
More Event Examples
// Mouse move element.addEventListener( "mousemove", trackMouse );
// Form submit form.addEventListener( "submit", handleSubmit );
// Page loaded window.addEventListener( "load", initPage );
