What are Conditionals

In computer programming, conditionals are programming constructs that let the program make decisions. They perform different actions or return different values based on whether a condition is true or false

Fun Fact!

Conditionals are like the "choose your own adventure" books - the story changes based on your decisions, just like code changes based on conditions!

Types of Conditionals:

if statement

The if statement is the most basic form of conditional statement. It checks if a condition is true. If it is, the program executes a block of code.

if (age > 18) { console.log("Adult"); }
if else statement

The if-else statement extends the if statement by adding an else clause. If the condition is false, the program executes the code in the else block.

if else (age == 18) { console.log("Just became an adult!") }
else statement

The else statement is used when the condition in the if and if else is not true.

else { console.log("Minor"); }
Complete Exampple:
let age = 18; if (age > 18) { console.log("Adult"); } if else (age == 18) { console.log("Just beacmae an adult!"); } else { console.log("Minor"); }
