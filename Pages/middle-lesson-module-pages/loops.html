What are Loops?

In computer programming, a loop is a sequence of instructions that is continually repeated until a certain condition is reached. Loops allow us to repeat a block of code multiple times. This is useful when you want to perform a task repeatedly without writing the same code over and over.

Real-World Analogy

Loops are like your morning routine - you repeat the same steps (brush teeth, get dressed, eat breakfast) every day, but with small variations.

For Loops

Use for loops when you know exactly how many times you want to repeat something:

JavaScript
// Count from 1 to 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// Loop through array items
const planets = ["Mercury", "Venus", "Earth"];
for (let i = 0; i < planets.length; i++) {
    console.log(planets[i]);
}

// Modern for-of loop
for (const planet of planets) {
    console.log(planet);
}
Python
# Count from 1 to 5
for i in range(1, 6):
    print(i)

# Loop through list items
planets = ["Mercury", "Venus", "Earth"]
for planet in planets:
    print(planet)

# With index
for i, planet in enumerate(planets):
    print(f"Planet {i+1}: {planet}"")

While Loops

Use while loops when you want to repeat until a certain condition changes:

JavaScript
 // Countdown from 5
let count = 5;
while (count > 0) {
    console.log(count);
    count--;
}

// User input validation
let userInput;
while (true) {
    userInput = prompt("Enter yes or no");
    if (userInput === "yes" || userInput === "no") {
        break;
    }
    console.log("Invalid input!");
}
Python
# Countdown from 5
count = 5
while count > 0:
    print(count)
    count -= 1

# User input validation
while True:
    user_input = input("Enter yes or no: ")
    if user_input in ["yes", "no"]:
        break
    print("Invalid input!")

Loop Control Statements

These statements change how loops execute:

Statement JavaScript Python Description
Break break; break Exit the loop immediately
Continue continue; continue Skip to the next iteration
Else N/A else: Python-only: runs if loop completes normally (no break)

Nested Loops

You can put loops inside other loops for complex patterns:

JavaScript
 // Multiplication table
for (let i = 1; i <=5; i++) {
    let row = "";
    for (let j = 1; j <=5; j++) {
        row += (i * j) + "\t";
    }
    console.log(row);
}
Python
# Multiplication table
for i in range(1, 6):
    for j in range(1, 6):
        print(i * j, end="\t")
    print()
Coding Challenge

Create a program that:

  1. Prints numbers 1 to 100
  2. For multiples of 3, print "Fizz" instead
  3. For multiples of 5, print "Buzz" instead
  4. For multiples of both 3 and 5, print "FizzBuzz"
JavaScript Solution
Python Solution
// JavaScript Solution
for (let i = 1; i <= 100; i++) {
    if (i % 15 === 0) {
        console.log("FizzBuzz");
    } else if (i % 3 === 0) {
        console.log("Fizz");
    } else if (i % 5 === 0) {
        console.log("Buzz");
    } else {
        console.log(i);
    }
}
# Python Solution
for i in range(1, 101):
    if i % 15 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)

Key Takeaways

  • Use for loops when you know how many iteration you need
  • Use while loops when you need to loop until a condition changes
  • break exits a loop immediately
  • continue skips to the next iteration
  • Nested loops are powerful but can impact performance
