Conditional Statements

35 min

What are Conditionals?

In computer programming, conditionals allow your programs to make decisions and execute different code based on whether conditions are true or false.

Real-World Analogy

Conditionals are like traffic lights - your actions (stop, slow down, go) depend on the light's current state (red, yellow, green).

Basic Conditional Structures

JavaScript
// if statement
if (age >= 13) {
    console.log("Teenager");
}

// if-else statement
if (score >= 70) {
    console.log("Passing");
}else {
    console.log("Failing");
}

// if-else if-else
if (grade == 'A') {
    console.log("Excellent!");
} else if (grade == 'B') {
    console.log("Good job");
} else {
    console.log("Keep trying");
}
Python
# if statement
if age >= 13:
    print("Teenager")

# if-else statement
if score >= 70:
    print("Passing")
else:
    print("Failing")

# if-elif-else
if grade == 'A':
    print("Excellent!")
elif grade == 'B':
    print("Good job")
else: 
    print("Keep trying")

Comparison Operators

These operators compare values and return true or false:

Operator Name Example Result
== Equal to 5 == 5 true
!= Not equal 5 != 3 true
> Greater than 5 > 3 true
< Less than 5 < 3 false
>= Greater or equal 5 >= 5 true
<= Less or equal 5 <= 3 false

Logical Operators

Combine multiple conditions using logical operators:

JavaScript
// AND (both conditions true)
if (age > 12 && age < 20) {
    console.log("Teenager");
}

// OR (either condition true)
if (day === "Saturday" || day === "Sunday") {
    console.log("Weekend!");
}

// NOT (reverse the condition)
if (!isRaining) {
    console.log("No umbrella needed");
}
Python
# AND (both conditions true)
if age > 12 and age < 20:
    print("Teenager")

# OR (either condition true)
if day == "Saturday" or day == "Sunday":
    print("Weekend"!)

# NOT (reverse the condition)
if not is_raining:
    print("No umbrella needed")

Nested Conditionals

You can put conditionals inside other conditionals for more complex decisions:

JavaScript
if (gameOver) {
    if (score > highScore) {
        console.log("New high score!");
        highScore = score;
    } else {
        console.log("Try again!");
    }
} else {
    console.log("Keep playing...");
}
Python
if game_over:
    if score > high_score:
        print("New high score!")
        high_score = score
    else: 
        print("Try again!")
else:
    print("Keep playing...")
Coding Challenge

Create a program that recommends activities based on weather conditions:

  • If sunny and warm (>70°F/21°C), suggest "Go to the beach"
  • If sunny but cold, suggest "Go for a walk"
  • If raining and warm, suggest "Visit a museum"
  • If raining and cold, suggest "Stay home and read"
  • For any other condition, suggest "Check the weather again"
JavaScript Solution
Python Solution
// JavaScript Solution
if (isSunny && temperature > 70) {
    console.log("Go to the beach");
} else if (isSunny && temperature <= 70) {
    console.log("Go for a walk");
} else if (isRaining && temperature > 70) {
    console.log("Visit a museum");
} else if (isRaining && temperature <= 70) {
    console.log("Stay home and read");
} else {
    console.log("Check the weather again");
}
# Python Solution
if is_sunny and temperature > 21:
    print("Go to the beach")
elif is_sunny and temperature <= 21:
    print("Go for a walk")
elif is_raining and temperature > 21:
    print("Visit a museum")
elif is_raining and temperature <= 21:
    print("Stay home and read")
else:
    print("Check the weather again")

Key Takeaways

  • Conditionals let programs make decisions
  • Use comparison operators to evaluate conditions
  • Logical operators combine multiple conditions
  • Nested conditionals handle complex logic
