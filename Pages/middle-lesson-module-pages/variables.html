Variables and Data Types (Strings, Numbers and Boolean)

30 min

What are Variables?

In computer programming, variables are like containers that store information in a program. They have a name and hold different types of data that can change while the program runs. A variable can refer to anything from numbers and strings to objects, collections and pointers.

Real-World Analogy

Think of variables like labeled boxes in your room. The label is the variable name, and the contents are the value. You can change what's inside the box without changing the label.

Declaring Variables

Different programming languages have different ways to create variables:

JavaScript
// Declaring variables
let name = "Alex";
const age = 13;
var score = 95.5;
  • let - for variables that can change
  • const - for constant values
  • var - older way (avoid in new code)
Python
# Declaring Variables
name = "Alex"
age = 13
score = 95.5
  • Python doesn't need special keywords
  • Just assign with =
  • Variables can change type

Common Data Types

Programming languages have different types of data that variables can hold:

Data Type JavaScript Example Python Example Description
Number let age = 13; age = 13 Integer or decimal numbers
String let name = "Alex" name = "Alex" Text (in quotes)
Boolean let isStudent = true; is_student = True True or False values
Array/List let scores = [95, 87, 92]; scores = [95, 87, 92] Ordered collection of items
Object/Dictionary let student = {name: "Alex", grade:7}; student = {"name": "Alex", "grade": 7} Key-value pairs

Type Conversion

Sometimes you need to convert between different data types:

JavaScript
// String to Number
let str = "123";
let num = Number(str);

// Number to string
let num2 = 456;
let str2 = String(num2);

// Other conversions
Boolean(1); // true
String(true); // "true"
Python
# String to number
str = "123"
num = int(str)

# Number to string
num2 = 456
str2 = str(num2)

# Other conversions
bool(1) # True
str(True) # "True"
